# Excel Dashboard Project

## Overview

Este projeto apresenta um painel analítico interativo criado inteiramente no Excel. Ele foi criado para demonstrar habilidades em transformação de dados, design de KPIs e construção de painéis usando ferramentas nativas do Excel.

## Objetivo

Este dashboard explora o perfil de compradores de bicicletas, permitindo identificar padrões de renda, idade e deslocamento entre clientes que compraram ou não uma bicicleta. A interface oferece filtros dinâmicos para Marital Status, Region e Education.

## Estrutura do Dataset

O arquivo final (`dashboard_final.xlsx`) contém quatro abas:

1. **bike_buyers**
   Base de dados original utilizada no projeto.

2. **Working Sheet**
   Área de tratamento e preparação dos dados antes da análise.

3. **Pivot Table**
   Tabelas Dinâmicas usadas para agregação e cálculo dos indicadores.

4. **Dashboard**
   Visualização final com KPIs, gráficos e segmentadores.

## KPIs and Insights

O dashboard apresenta indicadores derivados diretamente das visualizações e segmentações disponíveis:

### **Principais KPIs**

* Comparação da **renda média** entre compradores e não compradores.
* Distribuição de **faixas etárias** de clientes e sua relação com compra de bicicleta.
* Análise do **tempo de deslocamento (commute)** por faixa etária e status de compra.

### **Insights por gráficos**

* **Avg Income Per Purchase**
  Compara a renda média por gênero, segmentada entre "Yes" (comprou) e "No" (não comprou).

* **Customer Age Brackets**
  Mostra quantos clientes existem em cada faixa etária, novamente diferenciando compradores e não compradores.

* **Customer Commute**
  Apresenta padrões de deslocamento organizados por faixas etárias, também separados em compradores e não compradores.

### **Filtros (Slicers)**

O usuário pode segmentar todas as análises por:

* **Marital Status**
* **Region**
* **Education**
